Report of the 3-months internship at Czech Technical University

Introduction

This is a report done by Alexis Carbillet. It described what I did during his 3 months internship at Czech Technical University between the 6th of May 2019 to the 2nd of August 2019.

Let’s recall the description of the internship: The subject will address the field of data processing using probably one of the languages : Python, SQL, Java, R or HTML.

Several subjects were offered to me upon my arrival. The one I chose concerns data visualization because it is an area that interests me greatly. In addition, I want to become a data scientist so it was an opportunity to have a first experience in this area. Data is retrieved from telecom networks throughout the Czech Republic. The goal is to make a program using the R language that allows to process this data. To do this, the use of different libraries such as kableExtra and bookdown as well as dynamic HTML is necessary. This work is done under the R studio environment and will use computer tools such as PuTTY, Gitlab and OpenVPN.

What is PuTTY?

PuTTY is a terminal emulator. Originally available only for Windows, it is now focused on various Unix platforms. As part of this internship, it allows to send files in the gitlab repository, through command lines.

What is OpenVPN?

OpenVPN is free software for creating a virtual private network. This makes it possible to create a virtual machine. From there, you can access RStudio and GitLab.

What is GitLab?

GitLab is a free and open source solution for hosting Git projects. This structure is comparable to giants such as GitHub and BitBucket. The files will be saved on this platform by command line. I learned basic command lines in order to work on this topic.

What is the kableExtra package?

The purpose of the kableExtra package is to help build complex tables and manipulate table styles. It imports the magrittr “%>%” pipe symbol and verbalizes all functions. It is therefore possible to add “layers” to a kable output in the same way as ggplot2 and plotly. We will use this R package for some databases used in the project realized during this internship.

What is the bookdown package?

The bookdown package is an open source R package that facilitates the writing of long form books and articles / reports with Rmarkdown. It can generate books and ebooks ready for printing from Rmarkdown documents. We will also use this R package for some databases used in the project realized during this internship, in addition to kableExtra.

What is the ggraph package?

It is an extension of the ggplot2 API designed for viewing graphics. It provides the same flexible approach for creating layer-by-layer graphics. It allows in our project to make circle-packing.

What is the dygraphs package?

The dygraphs package is an R interface to the dygraphs JavaScript charting library. It allows users to explore and interpret dense data sets. In our project, it is used in the template studied. It allows to create dynamic graphics about the variety chosen.
