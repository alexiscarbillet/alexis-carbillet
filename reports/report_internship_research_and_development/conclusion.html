Conclusion

The objective of simulating human behaviour during an emergency situation led us to use the GAMA development environment which allows us to process multi-agent systems. This environment allows us to observe the behaviour of the agents present in the store and more particularly the decisions they can make in these situations. The purpose of this simulation is to find more efficient evacuation solutions.

During this internship, I discovered multi-system agents and Gama platform with its language GAML.I also learnt some psychology concepts that are explained in the first part of this report. The code is not available because it is confidential. I also used the software AutoCAD in order to create an environment where the agents can evoluate during the simulation.

One of the most important improvement that could be done is to add machine learning. In that case, the agents could learn during the simulation and be more realistic.
